# Explain Metatate Decision

Use the `metatate` MCP tool `explain-why`.

Require either:

- `decision_id`
- `validation_id`

Explain the policy evidence, conditions, obligations, prohibitions, and next
steps in plain language. Keep the identifier in the final answer so the user
can audit the result later.
