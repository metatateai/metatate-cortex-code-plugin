# Inspect Governance Rules

Use the `metatate` MCP tool `inspect-governance-rules`.

Require a fully qualified `table_name` when it is not already clear. If the
user mentions one column, pass it as `column_name`; otherwise omit
`column_name` to inspect rules at the table level.

Summarize applicable policies, conditions, obligations, prohibitions,
jurisdiction or purpose constraints, and source rule references.
