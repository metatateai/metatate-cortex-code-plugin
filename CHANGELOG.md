# Changelog

## 0.1.1 - 2026-05-22

- Changed Cortex MCP setup to use role-restricted Snowflake PAT headers by
  default.
- Documented OAuth as a fallback only after verifying role-specific scope
  behavior.
- Updated the helper and validation checks for PAT-based Cortex MCP
  registration.

## 0.1.0 - 2026-05-22

- Initial public Cortex Code plugin for Metatate.
- Added Cortex plugin manifest, commands, skill, governance reviewer subagent,
  and customer setup documentation.
- Added helper to create user-level Cortex MCP configuration for the
  Snowflake-managed Metatate MCP server.
