# Metatate

Metatate is installed but inactive. Enable it when you want Cortex Code to use
Metatate governed data context, policy decisions, query validation, and release
review guidance through the Snowflake-managed MCP server.

After enabling the plugin, confirm the MCP server is configured:

```text
/mcp
```

If the `metatate` MCP server is missing, follow `docs/cortex-code-install.md`.
